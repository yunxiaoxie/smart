# Smart Frontal use Angular1.x ES6 with Webpack

## 前端工程简介

- 前端工程采用npm/webpack管理项目

## Install Dependencies
- npm install --install dependents to local node_modules dir.

## How to run
- npm start -- to start web server for dev.

### How to build
- npm run build


### 打包脚本模块

| 模块 | 是否包含 |
|------|---------|
| es6 transpiling  | Yes |
| css splitting    | Yes |
| js minification  | Yes |
| css minification | Yes |
| sass support     | Yes |
| less support     | no  |
| source map       | Yes |
| cache busting    | No  |
| js linting       | No  |
| js unit test     | No  |
